# LINUX-FUNDAMENTAL
Practice Assignment

📚 Overview

This repository documents my hands-on learning journey through the Linux Fundamentals practice assignment, completed as part of my cybersecurity training. The objective of this exercise was to build a solid foundation in Linux, one of the most essential operating systems used by cybersecurity professionals for system administration, penetration testing, incident response, and digital forensics. 

🎯 Learning Objectives

Throughout this assignment, I gained practical experience in:

Understanding the Linux operating system and its architecture.

Navigating the Linux file system using the command line.

Executing essential Linux commands for file and directory management.

Working with file permissions and ownership.

Searching for files and directories efficiently.

Understanding shell operators and command-line functionality.

Building confidence in using the Linux terminal for real-world administrative tasks.

💡 Why This Matters

Linux powers the majority of servers, cloud infrastructure, and cybersecurity tools. Developing strong Linux skills is essential for Security Operations Center (SOC) Analysts, Penetration Testers, Incident Responders, and GRC professionals who need to understand the environments they secure.
